# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in CodeVault, please report it responsibly.

**Do not open a public issue.**

Instead, please email the maintainers directly or use GitHub's private vulnerability reporting feature:

1. Go to the [Security tab](https://github.com/schwarz-svg/Idea-factory/security) of this repository
2. Click **"Report a vulnerability"**
3. Provide a detailed description of the vulnerability

We will acknowledge your report within 48 hours and work with you to understand and address the issue.

## Scope

This is currently a client-side-only application with no backend, authentication, or payment processing. However, we still take the following seriously:

- XSS (Cross-Site Scripting) vulnerabilities
- Dependency vulnerabilities
- Any issue that could affect users who fork and extend this project
